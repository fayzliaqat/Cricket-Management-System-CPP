# 🏏 Cricket Management & Entertainment Application

A structured C++ console application developed during my **1st Semester (2023)** to demonstrate core programming fundamentals, procedural logic, and file orchestration.

## 🚀 Key Modules & Features
1. **ODI CWC23 Insights:** Displays critical performance metrics of top batters, bowlers, and fielders from elite teams.
2. **Dynamic Team Registration:** Employs C++ File Handling (`<fstream>`) to register a custom 11-player squad, saving player performance metrics directly into external database files.
3. **Interactive Hand Cricket Game:** A complete game engine simulating a toss, batting, and bowling cycles using randomized computer selection logic.
4. **Pakistan Cricket Schedule Tracker:** Displays upcoming international matches, locations, and schedules for the calendar year.

## 📁 Project Structure & Data Files
This repository includes the source code along with the required backend `.txt` database files:
* `MAIN FAF PROJECT.cpp` - The core application logic.
* `PakistanSchedule.txt` - Match calendar dataset.
* `*TeamStats.txt` - Performance statistics for Pakistan, India, Australia, and England.

## 🛠️ Technical Toolkit
* **Language:** C++
* **File Handling (`<fstream>`):** Used for data persistence across sessions.
* **Data Structures:** Multi-dimensional Arrays and Vectors.
